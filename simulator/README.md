# 🚀 Nasdaq Top 50 Interactive Trading Simulator

이 시뮬레이터는 사용자가 **2026년 1월부터 과거 역사적 데이터를 기준**으로 나스닥 시총 상위 50개 종목을 직접 매매하고 모멘텀(Winner-Take-All) 및 평균회귀(Z-Score/RSI) 전략을 연습할 수 있도록 돕는 대화형 시뮬레이터 패키지입니다.

이 시스템은 특정 시점에 **미래 데이터를 참조하지 않도록 설계(Zero Look-Ahead Bias)**되어 있어 완벽히 실전과 같은 매매 의사결정을 훈련할 수 있습니다.

## 🛠️ 실행 방법

이 시뮬레이터는 두 가지 실행 환경(웹 대시보드 버전 및 터미널 CLI 버전)을 모두 지원합니다.

### 1. 🏆 (추천) 웹 대시보드 버전 실행
웹 브라우저를 통해 실제 퀀트 거래소 터미널처럼 시각화된 차트와 대시보드 화면을 보며 마우스 클릭만으로 매매를 모의 연습할 수 있습니다.

```bash
# 터미널에서 실행 (FastAPI 백엔드 + React 프론트엔드가 동시에 실행되며 Ctrl+C로 종료)
./simulator/start_web.sh
```
서버가 정상 기동되면 자동으로 **`http://localhost:5173`** 포트로 프론트엔드가 열립니다. 브라우저로 접속해 투자 연습을 시작하세요!

### 2. 터미널 CLI 버전 실행
클래식한 터미널 텍스트 기반 시뮬레이터입니다. 키보드 입력만으로 빠르게 매매 연습을 끝내고 싶을 때 사용합니다.

```bash
python3 simulator/interactive_cli.py
```

## 🎮 시뮬레이터 명령어 목록

시뮬레이터가 켜지면 매 영업일 주식 매매 및 시간 여행을 다음의 간단한 명령어로 제어할 수 있습니다.

| 명령어 | 단축어 | 설명 |
| :--- | :---: | :--- |
| **`dashboard`** | `d` | 현재 시점의 시장 Breadth, WTA 주도주군 및 Top 15 스크리너 출력 |
| **`portfolio`** | `p` | 가상 현금, 보유 주식 잔고, 평균 매입 단가, 평가 수익률(PnL) 출력 |
| **`buy <ticker> <shares>`** | `b` | 지정한 수량만큼 종가 매수 실행 (거래 수수료 0.15% 가산) |
| **`sell <ticker> <shares>`** | `s` | 지정한 수량만큼 종가 매도 실행 (거래 수수료 0.15% 차감) |
| **`next`** | `n` | 시간 흐름을 **1 영업일** 뒤로 이동 (자산 가치 자동 재평가) |
| **`next_week`** | `nw` | 시간 흐름을 **5 영업일** 뒤로 이동 (1주일) |
| **`next_month`** | `nm` | 시간 흐름을 **20 영업일** 뒤로 이동 (1개월) |
| **`history`** | `h` | 과거 매수/매도 거래 이력 로그 확인 |
| **`help`** | - | 도움말 및 명령어 리스트 보기 |
| **`quit`** | `q` | 시뮬레이션을 중단하고 최종 투자 보고서 출력 후 종료 |

## 📈 투자 지표 활용 팁 (연습용)

1. **승자독식 지수 (WTA Index)**:
   * 지표가 **50% 이상**인 구간(Extreme Winner-Take-All)에서는 모멘텀 상위 1~5위 주도주(`WTA-Champion` 및 `WTA-Leader`)에만 투자 비중을 집중하는 전략이 우월합니다.
   * 지표가 **50% 이하**인 넓은 시장 참여 구간에서는 중하위 종목들까지 고르게 매매하는 것이 유리합니다.
2. **역추세 평균회귀 (Mean Reversion)**:
   * Z-Score가 `-1.5` 이하이거나 RSI가 `30` 이하로 떨어져 **과매도 상태**인 낙폭과대주를 매수하고, Z-Score가 `0.0` (이동평균) 근처로 복귀할 때 청산하는 훈련을 할 수 있습니다.
